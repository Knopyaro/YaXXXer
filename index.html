Как это работает

1. Вставьте логин пользователя в поле для ввода... Profit.

2. Чтобы скопировать зашифрованный логин, нажмите на него, он автоматически скопируется в буфер обмена (можно скопировать и выделением).

При нажатии на крестик поле для ввода очищается, можно вставлять новый логин

То же самое работает и для поля со вводом номера телефона (поддерживаются номера формата +79876543210)

При нажатии на крестик поле для ввода очищается, можно вставлять новый номер телефона
